Dog & Cow

Crack the color code using clues — and press the right button combo: Dog or Cow?

Buttons 0: Dog & Cow is a minimalist logic puzzle game where every level hides a color pattern. Your only tools? Clues that reveal how many of each color are in the grid — and two buttons: Dog 🐶 and Cow 🐮. Tap the right sequence to match the clue and reveal the correct grid!

🎮 Game Features:

  • 🧩 Grid coloring puzzles based on numeric color clues
  • 🐾 Use Dog and Cow buttons to build a 3-digit combo
  • 🎯 Real-time preview updates as you input each code
  • 🧠 Brain-teasing logic with simple yet deep mechanics
  • 🎨 Clean visuals with vibrant color themes
  • 📴 Play anytime — no internet required

Whether you're a puzzle enthusiast or just love quirky logic challenges, Buttons 0: Dog & Cow delivers a fresh twist with just two buttons and infinite possibilities.
